This project demonstrates a binary classification model using Logistic Regression on the Breast Cancer Wisconsin dataset. The workflow includes loading the dataset into Pandas, splitting into training and testing sets, standardizing features, and fitting a logistic regression model.
The model is evaluated using key metrics such as precision, recall, ROC‑AUC, and confusion matrix, with visualizations like the ROC curve to show performance.

Features:
Pandas Integration – Dataset loaded and managed using Pandas DataFrames for clarity and flexibility.

Data Preprocessing – Train/test split with stratification and feature scaling using StandardScaler.

Logistic Regression Model – Simple yet powerful binary classifier trained with scikit‑learn.

Evaluation Metrics – Includes confusion matrix, precision, recall, and ROC‑AUC for comprehensive performance analysis.

Visualization – ROC curve plotted with Matplotlib to illustrate model performance.

Beginner‑Friendly Explanations – Clear summary of logistic regression, sigmoid function, and threshold tuning.

Customizable Threshold – Demonstrates how adjusting the decision threshold affects precision/recall trade‑off.
